# microbit-project
## Small Reaction Time Measurement for Two Micro with Best Time Saving

This project measures the reaction time of two players using two BBC Micro devices. After a random delay, a signal appears and both players must press a button as quickly as possible.
The Micro measure and compare the reaction times, then display the winner and save the best reaction time achieved. The project demonstrates event handling, wireless communication, timers, and data storage in Micro programming.
